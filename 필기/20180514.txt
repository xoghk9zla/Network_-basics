디지털 방송

- 음성 데이터 영상등의 방송신호를 디지털화 하고 처리한 후 디지털 방식의 전송 시스템으로 송신하고 수신하는 방송을 의미한다

- NTSC, ATSC(미국식)
  -> 이동통신이 안됨	

- CAS, DRM			-> 한번 살펴보라고 함

- 아날로그와의 차이점
  -> 아날로그는 하나의 전파에 하나의 영상만 전달가능하지만, 디지털은 복수의 영상이 전달될수있다.
  -> 화질, 음질 등이 좋음

- 아날로그 때는 많은 주파수를 사용, 디지털은 적은 주파수를 사용해 다양한 작업가능

- 지상파 방송
  -> 지상에 있는 방송 송출로 전파를 송출
  -> 470~806Mhz 사용
  -> 고화질의 방송
  -> 난시청해소의 확대
  -> 아날로그의 비해 출력이 적음

- 케이블 방송
  -> 동축 케이블이나 광케이블 등을 통해 프로그램을 각 가정이나 지역으로 내보내는 방송
  -> 채널 용량이 풍부
  -> 케이블을 통해 전송하기 떄문에 난시청지역이나 기후조건에 상관이 없음
  -> 지역성 또는 세분화된 시청

- 디지털 위성 방송
  -> 위성 안테나를 통해 방송을 시청
  -> 케이블을 깔기 어려운 곳에서 사용하기에 좋음
  -> 전국 동시 방송 가능
  -> 넓은 영역 방송가능
  -> 감쇠현상 지연시간이 길다
  
- 케이블 vs 위성
  -> 케이블 망을 기본, 위성을 기본
